# Digital Haat - Cow & Goat Management System

A professional beginner-friendly Flask project for managing cows and goats in a livestock haat/farm business.

## Main Features

- Dashboard with livestock, health, feeding, sales, and expense summary
- Add, view, edit, and delete Cow/Goat profiles
- Track health/vaccination/treatment records
- Track feeding records and feed cost
- Track business expenses
- Record sales and profit/loss
- Simple financial report page
- Professional responsive UI using HTML, CSS, and JavaScript
- SQLite database using SQLAlchemy
- Uploaded cow images used as background/visual design assets

## How to Run

1. Open the project folder in VS Code or your code editor.
2. Create a virtual environment:

```bash
python -m venv venv
```

3. Activate the virtual environment:

Windows:

```bash
venv\Scripts\activate
```

Mac/Linux:

```bash
source venv/bin/activate
```

4. Install requirements:

```bash
pip install -r requirements.txt
```

5. Run the project:

```bash
python app.py
```

6. Open this URL in your browser:

```text
http://127.0.0.1:5000
```

## Database

The app uses SQLite. A database file named `digital_haat.db` will be created automatically when the app runs.
